## 도입부

Telegram에서 `@nadaeri_bot`, `@brosclawbot`, `@bluebors_bot`과 대화한 적 있으신가요? 이 3개가 저에게 각각 다른 역할을 하고 있습니다.

- **비서실장**(@brosclawbot): 일정 관리, 아침 브리핑
- **나대리**(@nadaeri_bot): 빠른 1차 응답, 실무 보조
- **나과장**(@bluebors_bot): 블로그 글 작성

"AI 에이전트를 Telegram에서 쓰고 싶지만, 코딩은 못 합니다." 이 글은 **코딩 없이 Telegram 봇 3개를 세팅하는 방법**을 설명합니다.

## 왜 Telegram인가 — 무료이고, 간단하고, 어디서든 접근 📱

Telegram은 무료이고, BotFather이라는 공식 도구가 있어서 코딩 없이 봇을 만들 수 있습니다. 또한 **어디서든 접근**할 수 있어서 스마트폰, PC 상관없이 일정을 관리하고 메시지를 보낼 수 있습니다.

### BotFather으로 봇 만들기 — 코드 1줄도 안 필요

1. Telegram에서 **@BotFather**을 검색합니다
2. `/newbot` 명령을 보냅니다
3. 봇 이름을 입력합니다 (예: "나과장")
4. 봇 username을 입력합니다 (예: `nadaeri_bot`, `bluebors_bot` 등 — 끝에 `_bot` 필수)
5. **토큰**이 발급됩니다 — 이걸 복사해서 저장해둡니다

```
# 발급되는 토큰 형태
8446750683:AAHO...ABCdefGHI...
```

이것이 전부입니다. 코딩이 전혀 필요 없습니다.

---

## OpenClaw에 봇 등록하기 — 역할 부여 🌟

토큰을 발급받았으면, OpenClaw에 이 봇을 등록합니다.

```bash
# OpenClaw CLI로 봇 연결
openclaw telegram connect --bot nadaeri --token "8446750683:AAHO..."

# 에이전트 세션 확인
openclaw status
```

등록할 때 각 봇에 **역할을 부여**합니다:

```markdown
## @brosclawbot — 비서실장
- 역할: 일정 관리, 아침 브리핑, 전체 진행 추적
- 말투: 격식있고 정리된 보고
- 채널: Telegram

## @nadaeri_bot — 나대리
- 역할: 빠른 1차 응답, 실무 보조, 필요 시 확인 후 보고
- 말투: 짧고 명확, 존댓말
- 채널: Telegram

## @bluebors_bot — 나과장
- 역할: 블로그 글 작성, 본문 작성 전문
- 말투: 친근하지만 정보 밀도 높음
- 채널: Telegram
```

## 멀티 에이전트의 장점 — 왜 3개를 따로 운영하는가 ⚡

### 단일 봇의 한계

한 개의 봇에 모든 걸 시키면:
- 모든 기능이 **혼재**되어 품질 저하
- 처리 속도 **느려짐**
- 뭐가 어디서 문제인지 **추적 어려움**

### 멀티 봇의 장점

| 항목 | 단일 봇 | 멀티 봇 (3개) |
|---|---|---|
| 역할 분리 | ❌ 불가 | ✅ 역할별 최적화 |
| 응답 속도 | 느림 | 빠름 (병목 회피) |
| 품질 | 보통 | 각 역할 특화로 향상 |
| 설정 복잡도 | 낮음 | 높음 (3개 각각 설정) |

### 실제로 운영되는 하루

- **09:00** — 비서실장이 날씨 + 일정 브리핑을 텔레그램으로 전송
- **10:00** — 대표님이 나대리에 "블로그 도와줘 써줘" → 나대리가 확인 후 나과장에게 전달
- **10:05** — 나과장이 블로그 글 작성 → 비서실장이 검수
- **10:30** — 비서실장이 텔레그램으로 결과 URL 전송

## 아침 브리핑 자동화 — Cron 스케줄링 ⏰

매일 아침 9시에 자동으로 날씨와 일정을 보내도록 설정할 수 있습니다:

```yaml
# OpenClaw cron 설정
schedules:
  - name: morning-brief
    cron: "0 9 * * *"  # 매일 09:00
    agent: bros
    prompt: "아침 브리핑 실행: 날씨 확인 → 캘린더 확인 → Telegram으로 보고"
```

이렇게 설정하면 **매일 아침 9시**에 비서실장이 자동으로 날씨와 일정을 정리해서 텔레그램으로 보내줍니다.

## 마무리

Telegram 봇 3개를 세팅하면, 나만의 AI 팀이 완성됩니다.

- **일정/브리핑** → 비서실장에게 물어보세요
- **빠른 응답/실무 보조** → 나대리에게 물어보세요
- **블로그 글 작성** → 나과장에게 요청하세요

코딩이 필요 없는 이유는, BotFather이 기본적인 봇 생성을 도와주고, OpenClaw가 에이전트 역할 관리를 해주기 때문입니다. **설정만 해두면, 매일 아침 자동으로 움직이는 AI 팀**을 가질 수 있습니다.
