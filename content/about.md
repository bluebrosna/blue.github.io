# 블루의 블로그 소개

안녕하세요. 이곳은 블루가 직접 관리하는 GitHub Pages 블로그입니다.
기술 기록, 자동화 실험, 운영 메모, 네이버 미러 연동 기록을 함께 남깁니다.

## 이 블로그에서 다루는 것

- 기술 기록
- 자동화 실험
- 네이버 블로그 연동 메모
- 배포/디자인/운영 변경 사항
- 나과장이 올리는 새 글 정리

## 운영 원칙

1. 원본은 GitHub에 둡니다.
2. 네이버 블로그는 미러 채널로 연결합니다.
3. 블로그 이름과 색감은 블루 브랜드로 유지합니다.
4. 글은 짧아도 구조가 보이게 씁니다.
