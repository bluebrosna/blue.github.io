## 1️⃣ Shortcuts란?

Mac에 기본 설치된 **자동화 앱**입니다.

- ✅ 프로그래밍 필요 없음
- ✅ 드래그 앤 드롭으로 동작 연결
- ✅ Mac + iPhone 연동 가능
- ✅ 무료

**在哪里找到?**
- Launchpad → Shortcuts
- 또는 Spotlight 검색 (`Cmd + Space` → "Shortcuts")

---

## 2️⃣ 첫 번째 자동화: 스크린샷 자동 업로드

### Step 1: Shortcuts 열기

```
Launchpad → Shortcuts 앱 실행
```

### Step 2: 새 Shortcut 만들기

1. 우측 상단 **+** 버튼 클릭
2. 이름 입력: "스크린샷 자동 업로드"

### Step 3: 동작 추가

**추가할 동작:**
```
1. Folder Action Watcher (스크린샷 폴더 감시)
2. Get Contents of Folder (파일 목록 가져오기)
3. Upload to iCloud (또는 Cloud 서비스)
4. Show Notification (알림 표시)
```

### 완성!

이제 스크린샷 저장만 하면 **자동으로 Cloud에 업로드**됩니다!

## 3️⃣ 실전 자동화 5선

### 자동화 1: 출근 시 자동 실행

```
트리거: "출근" 위치 도착
↓ 
1. Slack 열기
2. 특정 채널 메시지 전송 ("출근했습니다!")
3. 캘린더 열기
4. 오늘 일정 알림
```

### 자동화 2: 회의 전 알림

```
트리거: 캘린더 이벤트 "회의" 5분 전
↓
1. 알림 표시 ("회의 5분 전입니다!")
2. 회의室 Zoom 링크 열기
3. 노트 앱 열기
```

### 자동화 3: 이메일 답장 템플릿

```
트리거: "답장 템플릿" 단축키 입력
↓
1. 선택 메뉴 표시:
   - 감사 인사
   - 일정 조율
   - 거절
   - 연기 요청
2. 선택한 텍스트 클립보드에 복사
```

### 자동화 4: 파일 정리

```
트리거: "整理 파일" 단축키
↓
1. Downloads 폴더 열기
2. 30일 이상 된 파일 → 휴지통
3. 큰 파일 (>100MB) → 외부 저장 알림
```

### 자동화 5: 퇴근 정리

```
트리거: "퇴근" 단축키
↓
1. 모든 앱 창 닫기
2. Desktop 정리
3. Downloads 폴더 정리
4. Slack 상태 "외출"으로 변경
5. 알림: "퇴근 준비 완료!"
```

## 4️⃣ 단축키 설정

자주 쓰는 자동화은 **keyboard shortcut**을 할당하세요!

### 방법

1. Shortcuts 앱에서自动化 선택
2. 우측 하단 **Details** 클릭
3. **Keyboard Shortcut** 입력
4. **Add** 클릭

### 추천 단축키

|自动化 | 단축키 |
|--------|--------|
| 출근 모드 | `⌘ + ⌥ + 1` |
| 퇴근 모드 | `⌘ + ⌥ + 2` |
| 이메일 답장 | `⌘ + ⌥ + E` |
| 파일 정리 | `⌘ + ⌥ + F` |
