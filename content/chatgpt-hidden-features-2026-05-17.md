## 대부분의 사람은 ChatGPT의 20%만 사용합니다

ChatGPT를 "검색 엔진 대신" 혹은 "질문하는 곳"으로만 사용하고 계신가요? 실제로 이 도구에는日常工作에 극적인 효율을 가져다주는 숨겨진 기능들이 많지만, 대부분의 사용자들은 그 일부만 활용하고 있습니다.

이 글에서는 **실제로 써보면 삶이 바뀌는** 5가지 숨겨진 기능을 소개합니다.

---

## 기능 1: 파일 분석 — PDF, 이미지, 코드 파일 뭐든 올려보세요

ChatGPT Plus 구독자라면 **파일 첨부 기능**을 활용할 수 있습니다.

**활용 시나리오:**
- **PDF 계약서** → 핵심 조항 3줄 요약
- **엑셀 데이터** → 추세 분석 + 그래프 제안
- **源代码** → 버그 이유 설명 + 수정 코드 제공
- **이미지 스크린샷** → UI 디자인 피드백 요청

> 💡 업로드 버튼은 채팅 입력창 왼쪽에 있습니다. 종이 문자, 구조图,diagram 등 뭐든 가능하다!

![파일 업로드 위치](https://images.unsplash.com/photo-1460925895917-afdab827c52f?w=800&q=80)

## 기능 2: 메모장(Canvas) — 편집기 없이 문서 작성과 수정

很多人不知道 ChatGPT에는 **Canvas**라는 문서 편집 기능이 내장되어 있습니다.

**기존 채팅 방식의 한계:**
- 긴 문서를 수정하면 이전 내용이 짜증스럽게 다시 생성됨
- 부분 수정이 불가능해 전체를 다시 요청해야 함

**Canvas 방식의 장점:**
- 문서의 특정 부분만 선택 → 직접 수정 요청
- 글자 수, 톤, 구조 등 세밀한 조정 가능
- 코드인 경우 문법 강조 + 직접 실행까지 가능

**사용법:** ChatGPT에서 `/canvas` 명령어 입력, 또는 GPT-4 with Canvas 선택

## 기능 3: 역할(Role) 설정 — 같은 질문이라도 답변의質がpletely 달라집니다

"당신은 10년 경력의 마케팅 Director입니다. 이 보고서를 검토해주세요."

이 한 문장을 앞에 붙이냐 마느냐だけで、답변의 깊이와 방향이 극적으로 달라집니다.

**효과적인 역할 설정 방법:**

| 수준 | 예시 | 효과 |
|------|------|------|
| **기본** | "당신은厨师입니다" | 답변 방향 변경 |
| **경력 설정** | "10년 경력의 퍼스널 트레이너로서" | 전문성 증가 |
| **상황 추가** | "초보자를 가르치는 입장에서, 3문장으로 설명해줘" | 초보 맞춤 설명 |
| **제약 조건** | "마케터 관점에서, 경쟁사 대비 강점을 중심으로" | 방향 구체화 |

## 기능 4: 대화 저장 & 검색 — 했던 대화를 또 활용하기

매번 같은 설명을 반복하고 계신가요? ChatGPT는 **대화 저장소(Chat History)** 기능을 제공합니다.

**유용한 활용법:**
- **프로젝트별 대화 묶기** → 각 프로젝트마다 대화 스레드 분리
- **자주 묻는 Q&A 저장** → 나중에 검색으로 즉時 확인
- **문서 초안 보관** → 수정 이력 관리

**꿀팁:** 대화 제목을 직접 수정하면 나중에 검색할 때 훨씬 편리합니다. 기본값은 "New chat"이므로 반드시 변경하세요!

## 기능 5: Custom Instructions — 매번 같은 설정 입력하지 않기

매번 "당신은 한국어로 답변해주세요"를 입력하기 지치셨나요?

**Custom Instructions 설정:**
1. 설정 → Personalization → Custom Instructions
2. "답변에 포함되어야 할 내용"에 기본 선호 사항 입력
3. "답변에 포함되지 말아야 할 내용"에 거부 요청 입력

**예시 설정:**
```
回答语言: 항상 한국어로 답변
回答风格: 전문적이지만易懂하게
포함 사항: 구체적인 예시와 단계
제외 사항: 불필요한 전문 용어 بدون 설명
```

이렇게 설정하면, 이후 모든 대화에서 매번 같은 지시을 반복할 필요가 없어집니다!

## 이 기능들을 결합한 실전 시나리오

**가장 강력한 조합:**

> "당신은 경험 많은 비즈니스 컨설턴트입니다. 이 PDF 파일(계약서 첨부)의 주요 리스크 3가지를 한국어로 설명해주세요. 이후 이 내용을 바탕으로 우리 팀이 논의해야 할 포인트를 5개 정리해주세요."

→ 파일 분석 + 역할 설정 + 한국어 출력 +_actionable한 권고

이 조합 하나로,以往 수시간이 걸리던 문서 검토가 **10분**으로 단축됩니다.
