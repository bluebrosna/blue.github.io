## Mac의 진짜 힘은 단축키와 자동화에 있습니다

Mac을 사용하면서 "저건 어떻게 하는 거지?"と思った 적이 있으신가요? 대부분의 사람들이 마우스를利用してクリックとドラッグ를 반복하지만, Mac에는それを一瞬で終わらせる 기능들이 숨어 있습니다.

이 글에서는 **Mac 자동화의 3단계**:

1. 기본 단축키 (今日から使える)
2. 자동 실행 앱 (Automator + Shortcuts)
3. 스크립트 기초 (괜찮아, 코딩 아니에요)

를 다루겠습니다.

---

## 1단계:今日から試せる 단축키 10選

### 범용 단축키 (모든 앱에서 사용)

| 단축키 | 동작 | 비유 |
|--------|------|------|
| **Cmd + C** | 복사 | 문서 복사 |
| **Cmd + V** | 붙여넣기 | 복사한 내용 붙여넣기 |
| **Cmd + Z** | 실행 취소 | "저번으로" 버튼 |
| **Cmd + Shift + Z** | 다시 실행 | "다음으로" 버튼 |
| **Cmd + Q** | 앱 종료 | 창 닫기가 아닌 完全終了 |
| **Cmd + W** | 창 닫기 | 현재 탭/창만 닫기 |

### Finder (파일 탐색기) 단축키

| 단축키 | 동작 |
|--------|------|
| **Cmd + N** | 새 Finder 창 |
| **Cmd + Shift + N** | 새 폴더 |
| **Cmd + Delete** | 파일을 휴지통으로 |
| **Cmd + Shift + Delete** | 휴지통 비우기 |
| **Space** | 파일 미리보기 (快速视图) |
| **Cmd + I** | 파일 정보 보기 |

###截图 단축키 (매우有用!)

| 단축키 | 동작 |
|--------|------|
| **Cmd + Shift + 3** | 전체 화면 캡처 |
| **Cmd + Shift + 4** | 영역 선택 캡처 |
| **Cmd + Shift + 5** | 캡처 도구 모음 열기 |
| **Cmd + Shift + 6** | Touch Bar 캡처 (해당モデルの場合) |

> 💡 **Tip:** 캡처 후 바로 편집하고 싶다면 **Ctrl + 단축키**를 누르세요. 클립보드에 복사되지 않고 바로 미리보기가 열립니다.

## 2단계: Automator와 Shortcuts로.no-code 자동화

Mac에는 **코딩 없이** 자동화를 만들 수 있는 도구가 두 개内置되어 있습니다.

### Automator: 반복 작업을 자동화

**사용 예시:**

1. **여러 이미지를 한 번에 리사이즈**
   - Automator 실행 → "변환 이미지 크기" 액션 선택
   - 원하는 크기 설정 → 이미지를 드래그하면 완료

2. **특정 단어가 포함된 파일 자동 이동**
   - 파일 목록 → 해당 단어 포함 여부 필터
   - 조건에 맞으면 지정 폴더로 이동

3. **PDF에서 텍스트 추출 후 이메일 전송**
   - PDF 추가 → 텍스트 추출 → 메일 앱으로 전송

**Automator 시작 방법:**
- Spotlight 검색 (Cmd + Space) → "Automator" 입력
- 또는 applications 폴더에서 찾기

---

### Shortcuts (快捷键): iPhone/Mac 연동 자동화

**최근 Mac에 기본 설치된 Shortcuts 앱으로:**

| 자동화 예시 | 설명 |
|------------|------|
| "출근 시" → Wi-Fi 자동 전환 | 집을 벗어나면 데이터로 |
| 특정 앱 열기 | 하루 중 정해진 시간에 지정 앱 실행 |
| 클립보드 내용 자동 정리 | 복사한 텍스트에서 불필요한 공백 제거 |
| 파일 자동 백업 | 특정 폴더가 변경되면 외부 저장소로 복사 |

**Shortcuts 생성법:**
1. Shortcuts 앱 실행
2. 우측 하단 "+" 버튼
3. 작업 추가 (액션 검색 가능)
4. 실행 조건 설정 (시간/앱 실행 시)

## 3단계: 간단한 스크립트 (코딩 같지만 아닙니다)

### AppleScript: 영어로 명령을 내리는 언어

Mac에만 있는 **AppleScript**는 영어 문장을 쓰면 그것대로 동작하는 스크립트 언어입니다.

**예시 1:Finderで現在のフォルダを開く**
```applescript
tell application "Finder"
    activate
    make new Finder window
end tell
```

**예시 2: 특정 앱을 실행하고 창 크기 조절**
```applescript
tell application "Safari"
    activate
    set bounds of window 1 to {100, 100, 800, 600}
end tell
```

**실행 방법:**
- 스크립트 편집기 (Spotlight → "Script Editor")에 붙여넣기 → 실행 버튼

---

### Shell Script: 터미널 명령어 모으기

조금 더进阶하고 싶다면, 터미널에서 사용하는 **Shell Script**를 알아두면 좋습니다.

**예시: 매일 아침 개발용 폴더를 여는 스크립트**
```bash
#!/bin/bash
cd ~/Developer/Project
open -a "Visual Studio Code"
```

이 파일을 `.sh`로 저장하고 실행 권한을 주면 언제든 더블클릭으로 같은 동작을 할 수 있습니다.

## 실전 조합: Automator + AppleScript

가장 강력한 조합은 Automator의GUI 방식과 AppleScript의 유연성을 결합하는 것입니다.

예를 들어:
1. Automator로 "파일 추가" 트리거 설정
2. 추가된 파일을 AppleScript로 처리 (이름 변경, 이동, 변환)
3. 결과물을 지정 위치에 저장

이 조합으로 **주말마다 반복하는 백업 작업**을 完全自動化했습니다.
